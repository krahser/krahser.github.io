Estoy aprendiendo a usar esta herramienta, ya que la quiero para realizar mi trabajo de tesis.

  • ¿Qué es ansible?

Una herramienta de administración de computadoras que utiliza ssh -es decir que no necesitamos instalar un agente en nuestros equipos- para poder configurarlos, desplegar aplicaciones y asegurarnos su estado

Mi organización para realizar algún trabajo:

  1. Identificar las tareas a realizar
  2. Analizar los módulos y roles disponibles (buscar en galaxy)
  3. Escribir el playbook o rol

Configuracion inicial

Como todas las herramientas que usamos a menudo, es bueno que guardemos su configuración. Para esto se trabajan los dotfiles. En este caso 1

En cuanto a mis preferencias para esta herramienta decidi configurar el 1 y el directorio donde se almacenan los 1.

  • ~/.ansible.cfg 
[defaults]
hostfile = <path de archivos de hosts>
roles_path = <path para almacenar los roles>
  • Archivo de hosts

La documentación explica muy bien como configurar esto, aunque quisiera agregar algo más.

Los distintos equipos los podemos clasificar en grupos, o sin grupo también esta bien. Pero si o si necesitamos que el host esté definido en este archivo. Este nombre de host es una conexión ssh. Por lo tanto si queremos personalizar la conexión solo necesitamos agregar el host en el ssh/config
